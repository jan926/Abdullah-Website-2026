SF Games PC Admin Panel Export

Deploy this folder on your admin domain.
Run: npm install
Run locally: npm run dev
Build: npm run build

Important: copy your Supabase values into .env on the admin domain.
This export does not include node_modules, dist, .git, or your real .env file.
The public site Admin navbar button has been removed from the main project, but the /admin route still exists for direct/admin-domain access.
